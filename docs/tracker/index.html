RT

Release Tracker

Config-driven release planning for any SaaS or software project

How It Works

1

Copy the starter config

Fork saas-starter.config.json and rename it for your project

2

Define your phases

Edit the JSON to add your phases, milestones, tasks, KPIs, and release gates

3

Track progress

Click checkboxes to cycle: pending → in progress → completed. Progress saves to localStorage

4

Export & share

Export progress as JSON, import on another machine, or share the config with your team
